# StreamCast Network – Projeto Completo (Scaffold)

Este repositório contém a estrutura inicial profissional do projeto **StreamCast Network**, incluindo:

- Backend Java + Spring Boot
- Frontend React
- Bot do Discord
- Banco PostgreSQL via Docker
- docker-compose para orquestração
- Documentação básica e scripts

Este é um *scaffold*: a base para você evoluir o projeto real.
